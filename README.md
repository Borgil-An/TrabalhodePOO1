# TrabalhodePOO1
trabalho de poo1 do aluno Gustavo Reginatto da turma M2.

Códigos parte das classes foram dispobonilizadas pelo professor docente da matéria e modificados pelo aluno para se adequar á necessidade do mesmo.

1 - O aplicativo deve:
a. Deve ser provido um método para inserir novos lembretes. O método devereceber uma data e um lembrete (String).
b. Excluir um elemento a partir de um índice.
c. Buscar lembretes que contêm uma determinada substring, retornando-os a partir do método de busca (retornar um objeto List<Lembrete>).
d. Buscar por lembretes de um determinado mês e retorná-los (retornar um objeto List<Lembrete>).
e. Buscar por lembretes de um determinado dia e retorná-los (retornar um objeto List<Lembrete>).
f. Imprimir todos os lembretes.
g. Imprimir todos os lembretes de um determinado mês (mostrar em ordem crescente).
h. Imprimir todos os lembretes de um determinado dia (mostrar em ordem crescente).
i. Ordenar os lembretes em ordem crescente.

E esté codigo cumpre tudo que é requesitado para a realização do trabalho.
